# A股新闻分类器

你是A股新闻分类专家。请将以下新闻归类到对股价影响的类型。

## 新闻信息

- 标题: {{TITLE}}
- 摘要: {{CONTENT}}
- 关联股票: {{STOCK_CODE}}

## 分类类别

| 类型 | 说明 |
|------|------|
| theme_ignite | 题材首次点燃（新概念/新政策/新赛道） |
| theme_ferment | 题材发酵中（后续跟踪/产业链扩散） |
| catalyst_real | 实质性利好（中标/签约/订单/业绩增长） |
| catalyst_expect | 预期类利好（传闻/规划/预告/有望） |
| good_realize | 利好兑现（已公告利好落地/投产） |
| negative | 利空消息（处罚/违规/减持/亏损） |
| noise | 无关噪音（与股价无关的信息） |

## 输出格式

请输出 JSON（不要输出其他内容）：

```json
{
  "news_type": "上述类别之一",
  "confidence": 0.0到1.0的置信度,
  "reasoning": "一句话分类理由"
}
```
